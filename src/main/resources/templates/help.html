Installation

Project is built using Maven, so you should have JRE and Maven(or use Maven wrapper) installed to use the application.

Launching

You can run the application using ./mvnw spring-boot:run. Or you can build the JAR file with ./mvnw clean package. Then you can run the JAR file:

java -jar target/UrlShortener-0.0.1-SNAPSHOT.jar

Usage

On the initial use, you can login with username test and password test.

Application domain is set in global.properties file, property domain.
